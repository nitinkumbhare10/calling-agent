# Intel Developer Cloud Deployment Guide 🚀

This guide explains how to host your **Calling Agent (Python)** and **Dashboard (Next.js)** on the **Intel Tiber Developer Cloud** (DevCloud) instance you selected.

---

## 🛠️ Step 1: Launch the Instance in Intel Cloud
Based on your screenshot, you have selected:
* **Configuration:** AI PC
* **Instance Type:** `BM-HPARL Intel® Core™ Ultra 7 Processor 265T on HP` (Bare Metal, Free)

### To launch it:
1. **Instance Name:** Enter a name (e.g., `calling-agent-server`).
2. **SSH Key:** 
   * If you don't have an SSH key in the list, click **"Add SSH Key"** or go to the **SSH keys** tab at the top.
   * On your Windows laptop, open **PowerShell** and run:
     ```powershell
     ssh-keygen -t ed25519 -f "$env:USERPROFILE\.ssh\intel_devcloud"
     ```
     Press **Enter** for all prompts (no passphrase needed).
   * Open the generated public key in Notepad:
     ```powershell
     notepad "$env:USERPROFILE\.ssh\intel_devcloud.pub"
     ```
   * Copy the entire text and paste it into the Intel Cloud **SSH Key** section.
3. Click **Launch** / **Request** at the bottom of the page.
4. Go to the **Instances** tab and wait for your instance status to show `Active`. Copy the **IP Address** of your instance.

---

## 🔑 Step 2: Connect to the Instance via SSH
Once active, open **PowerShell** on your local machine and run:
```powershell
ssh -i ~/.ssh/intel_devcloud devcloud@<YOUR_INSTANCE_IP>
```
*(Replace `<YOUR_INSTANCE_IP>` with the actual IP address shown in your Intel Cloud dashboard).*

---

## ⚙️ Step 3: Install Required Software on Server
Once you are logged into the server (which is running Ubuntu), run the following commands to install Node.js, Python, PM2, and other system dependencies:

```bash
# 1. Update system packages
sudo apt update && sudo apt upgrade -y

# 2. Install Python 3.11, pip, and virtualenv
sudo apt install python3.11 python3.11-venv python3-pip unzip wget build-essential git -y

# 3. Install Node.js & npm (for Next.js Dashboard)
curl -fsSL https://deb.nodesource.com/setup_18.x | sudo -E bash -
sudo apt install -y nodejs

# 4. Install PM2 globally to run services in the background
sudo npm install -g pm2
```

---

## 📁 Step 4: Transfer Code to the Server

Create the project folder on the server:
```bash
mkdir -p /home/devcloud/calling-agent
```

### Option A: Uploading via SCP (Recommended)
From your **local laptop PowerShell** (in the project root directory `C:\Users\Octopus\Desktop\Calling Agent`):
1. Zip the project files (excluding `venv` and `.git`):
   ```powershell
   Compress-Archive -Path .\* -DestinationPath .\calling-agent.zip -Force
   ```
2. Upload the zip file to the server:
   ```powershell
   scp -i ~/.ssh/intel_devcloud .\calling-agent.zip devcloud@<YOUR_INSTANCE_IP>:/home/devcloud/
   ```
3. Go back to your **server SSH terminal** and extract it:
   ```bash
   unzip -o /home/devcloud/calling-agent.zip -d /home/devcloud/calling-agent
   rm /home/devcloud/calling-agent.zip
   ```

---

## ⚙️ Step 5: Configure Environment Variables

1. **Python Agent `.env`**:
   On the server, open the `.env` file and configure it:
   ```bash
   cd /home/devcloud/calling-agent
   cp .env.example .env  # If not uploaded
   nano .env
   ```
   Fill in your:
   * `LIVEKIT_URL`, `LIVEKIT_API_KEY`, `LIVEKIT_SECRET`
   * `DEEPGRAM_API_KEY`, `GROQ_API_KEY`
   * `SIP_TRUNK_ID`, etc.
   *(Press `Ctrl + O` then `Enter` to save, and `Ctrl + X` to exit).*

2. **Next.js Dashboard `.env.local`**:
   ```bash
   cd /home/devcloud/calling-agent/dashboard
   cp .env.example .env.local  # If not uploaded
   nano .env.local
   ```
   Fill in the required dashboard environment variables.

## 🚀 Step 6: Start Services using PM2

PM2 ensures your Python agent and Next.js dashboard run continuously in the background and restart automatically if they crash.

### 1. Setup & Start Python Calling Agent
```bash
cd /home/devcloud/calling-agent

# Create a virtual environment and install dependencies
python3.11 -m venv venv
source venv/bin/activate
pip install -r requirements.txt

# Start the agent using PM2
pm2 start "python agent.py start" --name "calling-agent"

# Deactivate venv
deactivate
```

### 2. Setup & Start Dashboard (Next.js)
```bash
cd /home/devcloud/calling-agent/dashboard

# Install npm dependencies
npm install

# Build Next.js app for production
npm run build

# Start Next.js with PM2
pm2 start "npm start" --name "dashboard"
```

### 3. Save PM2 Process List
To make sure PM2 starts these services automatically if the server reboots:
```bash
pm2 save
pm2 startup
```
*(Copy and paste the command generated by `pm2 startup` to configure system startup).*

---

## 📊 Useful PM2 Commands to Monitor Your Apps
* **Check Status:** `pm2 status`
* **View Logs (Real-time):** `pm2 logs` or specific log: `pm2 logs calling-agent` or `pm2 logs dashboard`
* **Restart:** `pm2 restart calling-agent`
* **Stop:** `pm2 stop all`
